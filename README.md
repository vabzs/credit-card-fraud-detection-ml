# credit-card-fraud-detection-ml# Credit Card Fraud Detection

## Overview
Detecting fraudulent credit card transactions using ML on imbalanced data.

## Dataset
- Source: Credit Card Fraud Detection Dataset (Kaggle)

## Technologies
Python, Pandas, Scikit-Learn, XGBoost, SMOTE

## Steps
- Handling missing values
- SMOTE oversampling
- Training Random Forest & XGBoost
- Evaluating precision, recall, F1

## Results
- XGBoost achieved best fraud detection recall.
